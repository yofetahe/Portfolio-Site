FRONT END

Front-end is a term that involves the building of webpages and user interfaces for web-applications. It implements the structure, design, behavior, and animation of websites, web applications, or mobile apps.

BACK END

Back end development refers to the server side of an application and everything that communicates between the database and the browser.

TOOLS

A programming tool or software development tool is a computer program that software developers use to create, debug, maintain, or otherwise support other programs and applications.
